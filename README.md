# warehouse-accounting-system
# Система учета и списания материальных ценностей

**Вариант 28** — Учебная практика УП.02

## Описание проекта
Информационная система для автоматизации учета и списания материальных ценностей на складе организации.

## Структура репозитория
- case_task_1 — моделирование бизнес-процессов (IDEF0, IDEF3, DFD, UML)
- case_task_2 — архитектурное проектирование (ТЗ, прототипы, ресурсы, сроки)
- case_task_3 — графический интерфейс на C++
